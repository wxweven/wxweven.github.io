1. vim常用命令

  • h：左移一个字符
  • l：右移一个字符
  • k：下移一个字符
  • j：上移一个字符
  • 冒号后，set number：显示行号
  • shift + g：移动到最后一行
  • 1 + shift + g：移动到第一行
  • n + shift + g：移动到第 n 行
  • 0：移动到行首
  • $：移动到行尾
  • I：在行首插入
  • A：在行尾插入
  • i：在当前位置前插入
  • a：在当前位置后插入
  • o：在当前行之后插入一行
  • O：在当前行之前插入一行
  • /text：查找text，按 n 查找下一个，按 N 查找前一个
  • ?text：反向查找text，按 n 查找下一个，按 N 查找前一个
  • % s/old/new/g：将old替换成new，替换当前行的所有匹配项
  • Ctrl + f：向下滚动一屏
  • ctrl + b：向上滚动一屏
  • u：撤销
  • U：撤销对整行的操作
  • ctrl + r：重做，即该撤销的撤销
  • x：删除当前字符
  • dd：删除当前行
  • 10d：删除从当前行开始的10行
  • yy：复制当前行
  • p：在当前光标后粘贴，如果之前使用了yy复制了某一行，那么在当前行的下一行进行粘贴
  • 冒号后，wq：退出
  • 冒号后，q!：强制退出
